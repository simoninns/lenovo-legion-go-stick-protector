# Lenovo Legion Go Stick Protector
## Overview
This project provides a 3D printable stick protector for the Lenovo Legion Go.  The protector is a simple clip that secures the thumbsticks to prevent damage.

This project is GPLv3 open-source.

<img src="/images/1.jpg" width="800">

## Printing instructions
The model has been printed and tested on the Prusa MK3S/MMU2S printer using Prusament PETG filament. All parts fit the standard print bed.

### Recommended print settings

Print settings (for Prusa Slicer) are as follows:

* Material: PETG
* Layer: 0.20mm (Quality)
* Infill: 15%
* No supports required
